# VSUARCPROJECT
Данный репозиторий содержит исходный код для работы веб-сайта https://vsuarcproject.ru/, а также серверную часть для работы мобильного приложения для работы с хранилищем паспортов музейных объектов [ACS (Archeology Card Storage)](https://github.com/afn66666666/diploma).

Логика сервера написана на языке PHP. Сервер выполняет роль посредника в общениие между хранилищем и приложением. Основной функционал - извлечение данных из REST запросов с клиентской стороны (мобильного приложения), и дальнейшее составление и выполнение SQL-запросов с последующей отправкой ответа в виде https запроса. Подключение к БД осуществляется при помощи библиотеки PDO. 

Сервер предоставляет API для работы с хранилишем паспортов музейных объектов: загрузка документов из хранилища с параметрами для фильтрации, создание нового паспорта, редактирование данных, отображение мест раскопок с их координатами.

API, предоставленное сервером:

GET
+ /d_login?login={login}&password={password}
+ /d_load_cards?search_parameters={params}
+ /d_excavations
+ /d_user?id={id}
+ /d_material?id={id}
+ /d_organization?id={id}
  
POST
+ /d_create_card {card data, user id, status}
+ /d_edit_card {card data, user id, status}
+ /d_remove_card {id}
